# Markdown 全格式示例

这是一份**实时富文本**编辑器的演示文档。点击任意段落即可编辑，光标移开后立即渲染成富文本 —— 下面覆盖了 Markdown 的*绝大多数*语法。

## 一到六级标题

# 一级标题 H1

## 二级标题 H2

### 三级标题 H3

#### 四级标题 H4

##### 五级标题 H5

###### 六级标题 H6

## 文本样式

正文里可以有 **加粗**、*斜体*、***粗斜体***、~~删除线~~、<u>下划线</u>，以及行内代码 `const answer = 42`。还支持上标 x<sup>2</sup>、下标 H<sub>2</sub>O，和 emoji 😀 🎉 ✅。

转义与边界：带空格的星号不会被误当作强调 —— 2 * 3 = 6、4 * 5 = 20；需要字面标记时用反斜杠转义，如 \*星号\*、\_下划线\_、\`反引号\`。

> 这是一段引用，常用来写注释或补充说明。引用可以嵌套：
> > 第二层引用，语气更轻。
>
> —— 设计原则：读起来像一页纸，而不是一个应用。

## 列表

无序列表，支持多级缩进：

- 第一层项目
  - 第二层嵌套
    - 第三层嵌套
- 另一个第一层项目

有序列表：

1. 打开一篇文档
   1. 选择模板
      1. 空白文稿
      2. 从示例开始
   2. 命名并保存
2. 点击段落进入编辑，比如插入一段命令：
   ```bash

   ```
3. 移开光标即渲染

## 代码

行内代码：`npm install`、`git commit -m "msg"`。代码块带语言标注与复制按钮，超长行可在卡片内横向滚动：

```js
function greet(name) {
  // 和 Markdown 打个招呼 —— 下面这行很长，卡片右缘会渐隐提示可横向滚动
  const endpoint = "https://api.example.com/v2/markdown/render?theme=paper&inline=true&cache=false&pretty=false&token=demo";
  return `Hello, ${name}! (${endpoint})`;
}
console.log(greet("Markdown"));
```

```bash
# 安装并运行
npx -y @dev/cli@latest --version
```

## 表格

| 快捷键 | 功能 | 平台 |
| --- | --- | :---: |
| ⌘B | 加粗 | 全部 |
| ⌘I | 斜体 | 全部 |
| ⌘K | 命令面板 | 全部 |
| ⌘F | 查找替换 | 全部 |

列较多、一行放不下时，表格会出现横向滚动条，可左右拖动查看（下表就是一个例子）：

| 命令 | 默认快捷键 | 作用域 | 支持平台 | 可否自定义 | 说明 |
| --- | --- | --- | --- | :---: | --- |
| 加粗 | ⌘B | 行内选区 | macOS / Windows / Linux | 可以 | 包裹所选文字为 **粗体** |
| 斜体 | ⌘I | 行内选区 | macOS / Windows / Linux | 可以 | 包裹所选文字为 *斜体* |
| 行内代码 | ⌘E | 行内选区 | macOS / Windows / Linux | 暂不支持 | 包裹所选文字为 `代码` |
| 命令面板 | ⌘K | 全局 | macOS / Windows / Linux | 可以 | 搜索并跳转文档与命令 |
| 查找替换 | ⌘F | 当前文档 | macOS / Windows / Linux | 可以 | 支持正则、区分大小写与全词匹配 |
